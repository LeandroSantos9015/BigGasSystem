Vamos criar um sistema WinForms do zero (SQL Server + Dapper + RBAC + QuestPDF).

Você deve conduzir em ciclos:
1) Faça no máximo 7 perguntas (escopo, entidades, fluxos, permissões, relatórios, conexão).
2) Implemente uma etapa mínima compilável:
   - Infrastructure (DbConnectionFactory)
   - Auth básico (Login + sessão + autorização)
   - 1 entidade CRUD (tela + controller + repo)
3) Repetir: perguntar próximo bloco e implementar.

Regras: View sem regra; Controller orquestra; Repo parametrizado; QuestPDF isolado.
